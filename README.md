# QLD Tiny House Planner

A React-based planning tool for Queensland secondary dwellings and granny flats.

## 🚀 Live Demo

[**Try it on CodeSandbox**](https://codesandbox.io/s/qld-tiny-house-planner) (paste the code there)

Or view the [Live GitHub Pages Demo](https://morbas.github.io/hello-world/)

## Features

- 📍 Interactive map to measure property land area
- 🏗️ Multiple construction materials (timber, steel, modular, etc.)
- 💰 Real-time cost calculations
- 📅 Build timeline generator
- ✅ Permit & approval requirements
- 💵 Rental income projections

## Usage

```bash
npx create-react-app tiny-house-planner
cd tiny-house-planner
# Copy qld-tiny-house-planner.jsx to src/
# Update src/App.js to import and use the component
npm start
```

## Local Development

The component requires React 18+. No additional dependencies needed!
